This script can be used to calculate the diameter of a fullerene cage.
It gives the diameter of the fullerene cage in Angstroms and nanometers.
It can take the xyz file as a first argument and calculate its diameter.

The usage on bash is as follows:

python3 Diameter_of_fullerene.py Fullerene.xyz
